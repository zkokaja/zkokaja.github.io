# zkokaja.github.io

This is my personal website and blog hosted on GitHub Pages.

You can view the site at [zkokaja.github.io](zkokaja.github.io).

## Locally

Use bundler to manage ruby dependencies

    $ gem install bundler
    $ bundle install
    $ bundle exec jekyll serve
